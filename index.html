About RescueLink

RescueLink is a progressive web app (PWA) built to save lives. It connects individuals to emergency responders and tracks disasters using real-time data. Whether it’s an earthquake, flood, or fire, RescueLink ensures help finds you — fast.

Key Features

📍 Location-Based Help

Send an SOS with your real-time location so responders know exactly where you are.

🌎 Calamity Tracker

Get live earthquake and weather updates using trusted data from global sources.

⚡ Offline Ready

Install the app and stay prepared even during network interruptions or power outages.

Install RescueLink

📱 On Mobile (Android)

  1. Open RescueLink in your Chrome browser.
  2. Tap the ⋮ (three dots) in the top-right corner.
  3. Choose “Add to Home Screen”.
  4. Launch it from your home screen.

💻 On Desktop (Windows / macOS)

  1. Open RescueLink in Chrome or Edge.
  2. Click the Install icon in the address bar.
  3. Confirm installation — it will open in its own window.
